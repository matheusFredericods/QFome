QFome - Delivery App Interface

Projeto desenvolvido para colocar em prática conceitos avançados de **React** e **TypeScript**.

## 🚀 Tecnologias
- React
- TypeScript
- Tailwind CSS
- Lucide React (Ícones)
- Vite

## 🛠️ Funcionalidades
- [x] Menu Responsivo (Mobile/Desktop)
- [x] Hero Section com fundo dinâmico
- [x] Carrossel de pedidos "Drag-to-Scroll" com `useRef`
- [x] Listagem de restaurantes e benefícios com `.map()`
